# Registro de construcción del dataset

## Categorías
- Cartón
- Metal
- Orgánico
- Plástico
- Vidrio

## Fuentes
- Fotografías propias
- Datasets públicos con licencia

## Criterios de inclusión
- Imagen clara
- Un residuo principal
- Formato JPG o PNG
- Sin datos personales
- Categoría identificable

## Criterios de exclusión
- Imagen borrosa
- Imagen repetida
- Varias categorías mezcladas
- Archivo corrupto
- Residuo no identificable
